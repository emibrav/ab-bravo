# Dr. Bravo Emiliano · Abogado · Córdoba

Landing page profesional para estudio jurídico. Un solo archivo HTML, sin dependencias externas ni frameworks.

## Contenido

- Navegación sticky con menú hamburguesa en mobile
- Hero con botones de agenda y WhatsApp
- Barra de confianza
- 6 áreas de práctica: Civil, Comercial, Consumidor, Familia, Sucesiones, Planes de Ahorro
- Proceso en 4 pasos
- Formulario de contacto + botón directo a WhatsApp
- Botón flotante de WhatsApp
- Totalmente responsive (mobile, tablet, desktop)

## Cómo publicar gratis con GitHub Pages

1. Crear un repositorio nuevo en GitHub (ej: `dr-bravo-web`)
2. Subir `index.html` al repositorio
3. Ir a **Settings → Pages**
4. En *Source*, elegir `main` branch y carpeta `/ (root)`
5. Guardar — en unos segundos la web estará en:
   `https://tu-usuario.github.io/dr-bravo-web/`

## Dominio propio (opcional)

Si tenés o comprás un dominio (ej: `drbravoemiliano.com.ar`):

1. En tu proveedor de dominio, crear un registro DNS tipo `CNAME` apuntando a `tu-usuario.github.io`
2. En GitHub Pages (Settings → Pages → Custom domain), escribir tu dominio
3. Activar "Enforce HTTPS"
